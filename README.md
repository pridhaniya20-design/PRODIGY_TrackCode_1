# PRODIGY_WD_02

This project is a simple Tic-Tac-Toe web application built using HTML, CSS, and JavaScript. It allows two players to play the game by taking turns and placing X or O on a 3×3 grid. The application checks for winning conditions, declares the winner or a draw, and provides a reset button to restart the game. It is designed as a beginner-friendly project to understand basic web development concepts and game logic.
